================================================================================
TROUBLESHOOTING
================================================================================

Issue: "No module named 'src'"
Solution: Make sure you're running from the project root directory

Issue: "Invalid access token"
Solution: Generate new token at developers.facebook.com/tools/explorer/

Issue: "Rate limit exceeded"
Solution: Wait 1 hour or use multiple tokens

Issue: "Database locked"
Solution: SQLite doesn't support concurrent writes. Use PostgreSQL.

Issue: Empty results
Solution: Check if search terms match actual advertisers
